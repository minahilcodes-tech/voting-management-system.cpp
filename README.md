#  Voting Management System

A console-based **C++** application developed using file handling and programming fundamentals. The system verifies voter eligibility, prevents duplicate voting using unique IDs, records votes, and displays election results.

##  Features

- Voter registration
- Age verification (18+)
- Unique ID validation
- Prevent duplicate voting
- Cast votes for different parties
- Store voting records using text files
- Display voting results

##  Technologies

- C++
- File Handling
- Visual Studio

##  How to Run

1. Open the project in Visual Studio.
2. Build and run the project.
3. Enter voter details.
4. Cast your vote.
5. View the voting results.
